Shyft

Shyft is full stack a bike sharing marketplace.

The Frontend was built with React and Bootstrap, and is deployed to GitHub Pages. The API was built with Django and Postgres, and is deployed to Heroku.

Code-laborate

Code-laborate is a full stack application where users can post programming problems they're experiencing and other users can make comments on those posts with helpful solutions!

This app was entirely peer-programmed. The Frontend was built with React and Bootstrap, and is deployed to GitHub Pages. The API was built with Express and MongoDB, and is deployed to Heroku.

Spots

Spots is a full stack application designed to track where you've been and where you'd like to go. Users can create new Spots, update them, and track whether they've seen them. Use it for hiking spots, restaurants, or hidden peaceful places to meditate.

The Frontend was built with Javascript, jQuery, and Bootstrap, and is deployed to GitHub Pages. The API was built with Express and MongoDB, and deployed to Heroku.

Soothing Tic Tac Toe

Soothing Tic Tac Toe is a frontend application built with Javascript, jQuery, HTML, CSS, and Bootstrap, and is deployed to GitHub Pages. Users can play tic tac toe in a soothing UI.
